# 文件管理



Node的文件管理，本质是在操作系统API上的包装。 



## `fs` 模块



文件操作集中在FS模块

```tsx
import {...} from 'fs'
```



接下来为大家演示这部分能力:

讲解见视频，文档见官网文档：

https://nodejs.org/api/fs.html

- path

  - 工作路径
  - 路径解析

- readFile

  - 读取文本文件
  - 流读取
  - 举例：读取JSON文件

- writeFile

  - 写入文本文件
  - 流写入

- readdir

- stat
